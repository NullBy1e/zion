The Zion Project

Project Overview

The goal of the project is to merge anonymity with E2EE.

What is Zion?

Zion provides end-to-end encryption with anonymity to create an environment for secure communication.

On the offical matrix website you can more find information about the protocol and supported clients.
matrix.org

How?

The basis of the project is the matrix protocol.

Zion is a matrix homeserver hosted as an onion service.

It is an open-source standard that use HTTP in json format for communication (XMPP uses XML)

Why?

Hosting matrix homeserver as an onion service creates a additional protection for the end user.

It makes it so that you can feel safe that the traffic that I receive can't be linked to your identity.

It also makes the server more secure, because I can't know what you have written on the server.

Matrix protocol allows you to create your own homeserver and still communicate with the main network(how?)
